# Repository Structure & Concepts Covered

The code files in this repository are systematically organized into structural modules to follow a logical learning progression:

### 📁 01_Basics_and_Control_Flow
Focuses on fundamental syntax, building logical blocks, and managing program execution execution pathways.
- **Python Basics:** Syntax, variables, data types, type conversion, user input, print formatting, operators, and comments.
- **Control Flow:** `if`, `elif`, and `else` statements, nested conditions, match-case pattern matching, `for` and `while` loops, break/continue statements, and loop-based problem-solving.

### 📁 02_Data_Structures
Focuses on storing, organizing, and manipulating data efficiently using Python's built-in collection types.
- **Strings:** Indexing, slicing, string methods, advanced validation, and string manipulation.
- **Lists & Tuples:** Creating, modifying, sorting, reversing, searching, updating data, and handling immutable collections.
- **Advanced Collections:** Dictionaries and sets fundamentals, along with robust data structure exception handling.

### 📁 03_Functions_and_OOP
Focuses on writing clean, reusable, dry, and scalable software architectures.
- **Functions:** Scoping, parameters and arguments, return values, default arguments, and recursive algorithms.
- **Functional Programming:** Lambda functions, `map()`, `filter()`, `reduce()`, and decorators.
- **Object-Oriented Programming (OOP):** Classes, objects, constructors, variable/method scopes (instance, class, static), inheritance, method overriding, encapsulation, properties, and operator overloading.

### 📁 04_Modules_and_Automation
Focuses on working with native operating system environments and modular code structures.
- **Python Modules:** Custom module generation, import handling, and core standard libraries (`math`, `random`, `time`, `os`).
- **File & System Automation:** Automated directory creation, managing files/folders, path validation, and system environment variables.
